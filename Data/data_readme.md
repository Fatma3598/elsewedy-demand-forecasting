# 📊 Data

This folder contains the datasets used across all notebooks.

| File | Description |
|------|--------------|
| `elsewedy_materials_dataset.csv` | **Baseline dataset** — 612 records covering 17 materials over 36 months (Jan 2022 - Dec 2024). Used for primary model training and evaluation. |
| `elsewedy_materials_dataset_expanded.csv` | **Expanded dataset** — 5-year version used to validate model robustness and stability at larger scale. |

## Schema (20 columns)

| Column | Description |
|--------|--------------|
| `Date` | Monthly record date |
| `Item_Code` | Unique material identifier |
| `Item_Description` | Human-readable material name |
| `Supplier` | Supplier name |
| `Purchase_Order_No` | PO reference number |
| `Document_Number` | Internal document reference |
| `Historical_Demand_Quantity (TON)` | Actual demand for that month |
| `Units_Sold/Used (TON)` | Units consumed |
| `Current_Stock_Level (TON)` | Inventory on hand |
| `Safety_Stock_Target (TON)` | Target safety buffer |
| `Material_Type` | Category (XLPE, PE, PVC, LSF, GSW, GST, Copper Screen, Aluminum Screen, Mica Tape, Water Blocking Tape, Semiconductor) |
| `Voltage_Level` | LV / MV / HV |
| `Supplier_Country_Of_Origin` | Supplier's country |
| `Price_Per_Unit (USD)` | Price per ton |
| `Delivery_Lead_Time (Days)` | Supplier lead time |
| `Payment_Terms` | Net 30/45/60/90 or Annual Contract |
| `Market_Trends (Index)` | Market index (baseline = 100) |
| `Number_Of_Alternative_Suppliers` | Count of backup suppliers |
| `Supplier_Performance_Score (1-10)` | Supplier reliability rating |
| `Risk_Score (1-10)` | Calculated supply chain risk |

## Notes

- Data is **synthetic**, generated to reflect realistic cable manufacturing procurement patterns based on domain input from Elsewedy Electric.
- Generated by `notebooks/ElSewedy_Dataset.ipynb` — re-run that notebook to regenerate or modify these files.
